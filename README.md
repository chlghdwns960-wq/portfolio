# Choi Hongjun Portfolio

웹 퍼블리셔 / 웹디자이너 취업 지원을 위한 개인 포트폴리오 사이트입니다.  
반응형 화면 구현, 사용자 흐름 정리, 프로젝트별 결과물 소개, GitHub Pages 배포까지 하나의 웹 포트폴리오로 구성했습니다.

## Live Demo

https://chlghdwns960-wq.github.io/portfolio/

## Role

- 포트폴리오 전체 정보 구조 설계
- 반응형 레이아웃 및 인터랙션 구현
- 프로젝트별 핵심 역할, 기술, 판단 기준 정리
- 상세페이지 원본 보기 및 케이스 페이지 구성
- GitHub Pages 배포 및 저장소 관리

## Featured Projects

- **PetLog Mobile Web App**  
  반려동물 기록, 사진, 일정, 병원 정보를 모바일 앱 흐름으로 구성한 웹앱 프로젝트

- **Zipbanchan Responsive Renewal**  
  커머스 화면을 PC와 Mobile 기준으로 재구성한 반응형 웹 리뉴얼 프로젝트

- **Pocketwalk Branding & Detail Page**  
  반려견 산책용 제품 콘셉트, 브랜드 무드, 상세페이지 CTA 흐름을 정리한 브랜딩 프로젝트

- **E-commerce Detail Page Design**  
  제품 USP, 구매 불안 요소, 후기, 인증 정보를 구매 설득 흐름으로 구성한 상세페이지 디자인 프로젝트

이 포트폴리오는 단순히 결과 이미지를 나열하기보다, 각 프로젝트에서 어떤 흐름을 설계했고 어떤 기준으로 화면을 구성했는지 보여주는 데 초점을 두었습니다.
